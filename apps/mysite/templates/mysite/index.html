Ditty

Ditty is a one-stop shop for all of your favorite Music Festivals where you can learn about the artists, sample their music, and build a personal calendar for all of the shows you want to see. This was built on the MEAN stack using MongoDB to store the festival and artist information, as well as your personal calendar. Login and registration is encrypted using bcrypt, and there is also the option of using Facebook login.

  • Date: August 2016

Groups

Groups is a social network site designed to encourage inclusive communities. Users can post messages on other user's profiles, and comment on posted messages. Login and registration is encrypted using bcrypt. This was built on the Python stack using the Django framework and SQLite3 for storing user information and all messages/comments. The UI was built with Bootstrap.

  • Date: July 2016

Hint.e

Hint.e is a social iPhone game similar to Taboo. Randomly generated movie titles will show on the screen, and one player needs to get a teammate to guess the movie title without using any of the words in the title. This was built in Swift utilizing CoreData to store player's names and tracking score. AVPlayer was used to provide in-game music and sound effects. This was built in a single day hackathon.

  • Date: September 2016

Pokémon Stay

In response to the PokémonGo craze, my fellow classmates and I decided to make a simple Pokémon battle game where you Stay at your computer. Select a team of 3 Pokémon characters to battle against AI. As you level-up you gain access to stronger Pokémon to choose for your team. This was built in Python on a Django framework. We accessed the PokeAPI using jQuery. I built the backend on MySQL to store player and team information.

  • Date: July 2016

Clust 'r Bust

Clust 'r Bust is a multi-player iPhone app under development. The goal is to score points by matching colored disks. You can earn a power-up which allows you to fling a disk, or disks, off of your screen which is worth extra points. This was built with the Swift language using SpriteKit, and with socket.io to connect with multiple players. I built a server with Node.js to host the app.

  • Date: September 2016

Elimination Challenge

Elimination Challenge was a single-day hackathon project to create a multi-player iPhone app. There are 4 different challenges and players take turns trying to complete them within the given time. With each success, the challenges grow more difficult. This was written in Swift.

  • Date: September 2016
